# Shipping Partner Data Collection

A React-based web application for collecting and managing shipping partner information.

## Features

* Shipping Partner Login
* Profile Dashboard
* Aadhaar Details Collection
* PAN Details Collection
* Driving License Information
* Profile Data Management
* Simple and Responsive User Interface

## Technologies Used

* React.js
* JavaScript
* HTML
* CSS

## Getting Started

1. Clone the repository.
2. Install dependencies using `npm install`.
3. Start the application using `npm start`.

## Purpose

This project was developed to demonstrate a basic React application for securely collecting and displaying shipping partner profile information.
